# Psychology Patterns Index (Static Site)

A contributor-friendly static website that renders a searchable index of psychology patterns from `patterns.tsv`.

## Run locally

Use any local static server from this folder, for example:

```bash
python3 -m http.server
```

Then open `http://localhost:8000`.

## Update content

Edit `patterns.tsv` and keep it tab-separated with the same header row.
